# La maison dentaire

Site vitrine de La maison dentaire — soins dentaires d'excellence à Strasbourg.

## Technologies

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Installation

```sh
npm install
npm run dev
```

```sh
npm run build
```

Les fichiers de production seront générés dans le dossier `dist/`.
